## Contenido por Día

### Semana 1: Fundamentos ✅
- **Día 1:** Mentalidad IA-First - Marco de decisión para elegir herramientas
- **Día 2:** Caso Comparativo CFDI 4.0 - Mismo problema con ChatGPT, Claude y Gemini
- **Día 3:** Prompts Universales - Fórmula ROL+CONTEXTO+TAREA+FORMATO + 10 plantillas

### Semana 2: Especialización ✅
- **Día 4:** Custom GPTs Fiscales - Asistentes especializados sin código (RESICO, nómina, CFDI)
- **Día 5:** Claude Projects - Análisis de contratos, dictámenes y normativa extensa
- **Día 6:** Gemini + Google Workspace - Conexión con Drive, Sheets, Docs

### Semana 3: Herramientas Avanzadas ⏳
- **Día 7:** NotebookLM - Notebooks con LISR, CFF y criterios SAT (RAG)
- **Día 8:** Perplexity Pro - Búsqueda en tiempo real con fuentes citadas
- **Día 9:** Workflow Definitivo - Flujos híbridos combinando herramientas

### Semana 4: Aplicación y Negocio ⏳
- **Día 10:** Caso Multimodal - Procesamiento de múltiples formatos, auditorías
- **Día 11:** Diseña tu Sitio Web - HTML/CSS sin programación, hosting gratuito
- **Día 12:** Monetización - 4 modelos de negocio con IA

## Estructura de Cada Día

Cada día incluye:
- `slides-script.md` - Presentación con script para teleprompter
- `ejercicios-practicos.md` - Ejercicios guiados (30-45 min)
- `README.md` - Resumen para Notion
- Recursos adicionales específicos del tema
