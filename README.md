# Reto 12 Días de IA para Contadores

Contenido completo del reto de 12 días de Inteligencia Artificial para contadores.

| Semana | Días | Enfoque |

| 2 | 4-6 | Especialización |
| 3 | 7-9 | Herramientas Avanzadas |
| 4 | 10-12 | Aplicación y Negocio |
