# Dia 8: Perplexity Pro

## Resumen

Perplexity es un buscador potenciado con IA que busca en internet, sintetiza multiples fuentes y te da respuestas directas con citas verificables. A diferencia de Google donde tu interpretas los resultados, Perplexity hace el trabajo de investigacion por ti y siempre te muestra exactamente de donde viene cada dato.

## Puntos Clave

- **Citas siempre:** Cada dato tiene fuente verificable con link
- **Tiempo real:** Busca en internet en cada consulta
- **Sintesis inteligente:** Resume multiples fuentes en una respuesta
- **Pro Search:** Analisis profundo para preguntas complejas
- **Focus modes:** Especializa busqueda (academico, noticias, YouTube)

## Casos de Uso para Contadores

1. Investigar reformas fiscales recientes con fuentes del DOF
2. Buscar criterios del SAT con citas verificables
3. Encontrar jurisprudencia y tesis fiscales
4. Comparar interpretaciones de diferentes fuentes
5. Verificar informacion con respaldo de fuentes
