# Dia 10: Caso Multimodal

## Resumen

El Dia 10 abre la Semana 4: Aplicacion y Negocio. Hasta ahora has trabajado con un tipo de input a la vez: texto, documentos o busquedas. Hoy aprenderas a combinar multiples formatos en un mismo proyecto: PDFs, imagenes escaneadas, hojas de calculo, estados de cuenta, facturas, contratos y mas. Este es el caso de uso real que enfrentas en auditorias, due diligence y consultoria compleja.

## Puntos Clave

- **Multimodalidad:** Las IAs modernas procesan texto, imagenes, PDFs y tablas
- **Casos reales:** Auditorias, due diligence, analisis de expedientes completos
- **Vision por computadora:** Extraer datos de documentos escaneados sin OCR manual
- **Integracion de fuentes:** Combinar informacion de multiples archivos
- **Herramientas especializadas:** Claude, ChatGPT y Gemini tienen capacidades diferentes

## Capacidades Multimodales por Herramienta

| Herramienta | PDFs | Imagenes | Hojas de calculo | Contexto largo |
|-------------|------|----------|------------------|----------------|
| ChatGPT | Si | Si | Si (via Code Interpreter) | 128K tokens |
| Claude | Si | Si | Si (analisis visual) | 200K tokens |
| Gemini | Si | Si | Si (integracion Sheets) | 1M+ tokens |

## Casos de Uso Tipicos

1. **Auditoria fiscal:** Estados financieros (Excel) + facturas (PDF) + comprobantes (imagenes)
2. **Due diligence:** Contratos (PDF) + estados de cuenta (PDF) + documentos corporativos (imagenes)
3. **Revision de expediente:** Dictamenes (PDF) + anexos (Excel) + documentacion soporte (imagenes)
4. **Analisis de cumplimiento:** Declaraciones (PDF) + comprobantes de pago (imagenes) + conciliaciones (Excel)
5. **Consultoria integral:** Documentos legales + informacion financiera + evidencia documental
