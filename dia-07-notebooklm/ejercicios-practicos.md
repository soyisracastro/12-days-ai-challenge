## Ejercicio 1: Crear tu Primer Notebook (15 minutos)

### Objetivo
Crear un notebook de legislacion fiscal y hacer tus primeras consultas.

### Paso 1: Acceder a NotebookLM (2 minutos)

1. Ve a https://notebooklm.google.com
2. Inicia sesion con tu cuenta de Google
3. Veras la pantalla principal con opcion de crear nuevo notebook

### Paso 2: Crear el Notebook (3 minutos)

1. Click en "New Notebook" o "Crear notebook"
2. **Nombre sugerido:** "Legislacion Fiscal Mexico - Consulta Rapida"
3. Veras la pantalla para agregar fuentes

### Paso 3: Subir Documentos (5 minutos)

1. Click en "Add source" o "Agregar fuente"
2. Selecciona "Upload" para subir PDFs
3. Sube al menos uno de estos documentos:
   - **Opcion A:** LISR completa (recomendado para este ejercicio)
   - **Opcion B:** CFF si prefieres
   - **Opcion C:** Un documento de tu trabajo (criterios, contratos)

4. Espera que NotebookLM procese el documento (1-2 minutos)

**Indicador de exito:** Veras el documento listado con un resumen automatico

### Paso 4: Primera Consulta (5 minutos)

Ahora haz tu primera pregunta. Escribe en el chat:

```
¿Cuales son los requisitos para que un gasto sea deducible segun esta ley?
```

**Evalua la respuesta:**
- [ ] NotebookLM respondio basandose en el documento
- [ ] Cito el articulo o seccion especifica
- [ ] Puedo hacer click en la cita para ver el original

**Segunda pregunta:**
```
Cita textualmente el articulo que habla sobre deducciones no autorizadas.
```

**Evalua:**
- [ ] Cito el texto exacto del documento
- [ ] La referencia es correcta

## Ejercicio 2: Preguntas Especificas con Citas (10 minutos)

### Objetivo
Practicar diferentes tipos de preguntas y verificar la precision de las citas.

### Instrucciones

Haz las siguientes preguntas (adapta segun el documento que subiste):

**Pregunta 1: Busqueda de articulo especifico**
```
¿Que dice el articulo 27 sobre requisitos de deducciones?
```

**Verifica:**
- [ ] La respuesta corresponde al articulo correcto
- [ ] Abre el documento original y compara

**Pregunta 2: Busqueda tematica**
```
Lista todos los gastos que NO son deducibles segun este documento.
```

**Verifica:**
- [ ] La lista es completa
- [ ] Cada punto tiene referencia

**Pregunta 3: Interpretacion**
```
Explica en terminos simples que tratamiento tiene el arrendamiento en esta ley.
```

**Verifica:**
- [ ] La explicacion es clara
- [ ] Se basa en el documento, no inventa

**Pregunta 4: Busqueda de termino**
```
¿En que partes de este documento se menciona "partes relacionadas"?
```

**Verifica:**
- [ ] Lista todas las menciones
- [ ] Puedes verificar cada una

**Pregunta 5: Que NO sabe**
```
¿Cual es la tasa de IVA vigente para 2024?
```

**Esperado:**
- NotebookLM deberia decir que no tiene esa informacion (si solo subiste LISR)
- Esto demuestra que NO inventa

**¿Todas las citas fueron correctas?** Si / No

## Problemas Comunes y Soluciones

### Problema 1: "NotebookLM no carga"
**Solucion:**
- Verifica que usas una cuenta Google valida
- Prueba con Chrome si usas otro navegador
- Limpia cache del navegador
- Revisa tu conexion a internet

### Problema 2: "El documento tarda mucho en procesarse"
**Solucion:**
- PDFs muy largos (200+ paginas) pueden tomar 5+ minutos
- Espera pacientemente, no cierres la ventana
- Si falla, intenta con un documento mas corto primero

### Problema 3: "Las respuestas no son precisas"
**Solucion:**
- Verifica que el documento se proceso correctamente
- Haz preguntas mas especificas
- Pide que cite exactamente donde encontro la informacion
- Si el PDF es escaneado (imagen), puede no leerse bien

### Problema 4: "Audio Overview no esta disponible"
**Solucion:**
- La funcion puede no estar disponible en todas las regiones
- Intenta con documentos en ingles si no funciona en espanol
- La funcion puede tener limite de uso diario

### Problema 5: "No tengo los PDFs de las leyes"
**Solucion:**
- Descargalos de diputados.gob.mx (links en requisitos)
- O usa cualquier documento PDF que tengas a la mano
- Puedes agregar URLs de sitios web en lugar de PDFs
