# Día 1: Mentalidad IA-First
## Reto 12 Días de IA para Contadores

---

## Resumen del Día

Hoy aprendimos a pensar diferente. La **Mentalidad IA-First** significa hacer una pausa antes de cualquier tarea y preguntarte: *¿puede la IA ayudarme con esto?*

No se trata de reemplazarte, se trata de trabajar más inteligente.

---

## Las 3 Herramientas Clave

| Herramienta | Fortaleza Principal | Cuándo Usarla |
|-------------|--------------------|--------------|
| **ChatGPT** | Versatilidad | Tareas generales, redacción, explicaciones, Custom GPTs |
| **Claude** | Profundidad | Documentos largos, análisis complejo, razonamiento legal |
| **Gemini** | Actualidad | Info en tiempo real, integración con Google Workspace |

---

## Framework de Decisión Rápida

Antes de abrir cualquier herramienta, hazte estas 3 preguntas:

1. **¿Qué tipo de tarea es?**
   - Redacción/General → ChatGPT
   - Análisis profundo → Claude
   - Búsqueda/Research → Gemini

2. **¿Necesito información actualizada (2024-2025)?**
   - Sí → Gemini o Perplexity
   - No → ChatGPT o Claude

3. **¿Qué tan largo/complejo es?**
   - Corto/Simple → ChatGPT
   - Largo/Complejo → Claude

---

## Fórmula del Prompt Efectivo

```
ROL + CONTEXTO + TAREA + FORMATO
```

**Ejemplo:**
```
[ROL] Actúa como contador fiscal especialista en México.
[CONTEXTO] Mi cliente es persona física RESICO del sector comercio.
[TAREA] Analiza si puede deducir gastos de gasolina de su auto personal.
[FORMATO] Responde con: 1) Si es deducible, 2) Requisitos, 3) Fundamento legal.
```

## Errores a Evitar

- Usar solo una herramienta para todo
- Confiar ciegamente sin verificar
- No dar contexto suficiente en los prompts
- Pedir información actualizada a ChatGPT/Claude
- Copiar y pegar respuestas sin revisar
