## Reto 12 Días de IA para Contadores

## Resumen del Día

Hoy aprendiste a usar **Claude Projects** para analizar documentos extremadamente largos y complejos. Mientras que los Custom GPTs de ayer son especialistas rápidos, Claude Projects es tu analista senior que lee 200 páginas y mantiene coherencia en todo el análisis.

No más leer contratos completos manualmente. No más buscar artículos en PDFs de 300 páginas. Claude Projects procesa, tú decides.

---

## ¿Qué es Claude Projects?

Claude Projects es un espacio de trabajo persistente en Claude donde puedes:

- Subir documentos de hasta **200 páginas**
- Crear una **base de conocimiento** que permanece disponible
- Hacer **conversaciones ilimitadas** con acceso a todos los documentos
- Agregar **instrucciones personalizadas** como en Custom GPTs
- Analizar **múltiples documentos simultáneamente**

**Es como crear una biblioteca especializada para cada caso o cliente.**

---

## Diferencias Clave: Custom GPTs vs Claude Projects

| Aspecto | Custom GPTs (ChatGPT) | Claude Projects |
|---------|----------------------|-----------------|
| **Fortaleza** | Respuestas rápidas y consistentes | Análisis profundo de docs largos |
| **Capacidad por doc** | 25-30 páginas | 200 páginas |
| **Cantidad de docs** | Hasta 20 archivos | 5-10 archivos (pero más largos) |
| **Compartir** | Fácil (link público) | Solo tú (por ahora) |
| **Uso ideal** | Consultas repetitivas | Análisis único y complejo |
| **Velocidad** | Instantánea | 30-60 seg procesamiento inicial |
| **Costo** | ChatGPT Plus (20 USD/mes) | Claude Pro (20 USD/mes) |

**Ambas son complementarias, no competidoras.**

## Casos de Uso para Contadores

### 1. Análisis de Contratos Largos
- **Documento:** Contrato de 80+ páginas (arrendamiento, franquicia, servicios)
- **Qué hace:** Identifica cláusulas con implicación fiscal, obligaciones y riesgos
- **Ahorra:** 2-4 horas de lectura manual → 30-45 minutos con Claude

### 2. Revisión de Dictámenes Fiscales
- **Documento:** Dictamen completo con anexos
- **Qué hace:** Encuentra inconsistencias, valida cifras, identifica riesgos
- **Ahorra:** 4-6 horas de revisión → 1-1.5 horas con Claude

### 3. Biblioteca Legal Personal
- **Documentos:** LISR + CFF + LIVA + RMF completas
- **Qué hace:** Consulta rápida de artículos sin buscar en PDFs
- **Ahorra:** 5-15 min por búsqueda → 30 segundos con Claude

### 4. Due Diligence Fiscal
- **Documentos:** Estados financieros + declaraciones + contratos + actas
- **Qué hace:** Análisis cruzado, identificación de riesgos, pasivos contingentes
- **Ahorra:** 2-4 semanas equipo completo → 1 semana con Claude

### 5. Comparación de Versiones
- **Documentos:** Contrato v1 vs v2 / Dictamen borrador vs final
- **Qué hace:** Identifica exactamente qué cambió y su impacto
- **Ahorra:** 1-3 horas → 10-20 minutos con Claude

---

## Los 5 Tipos de Preguntas Efectivas

### 1. Búsqueda Específica
```
¿Qué dice exactamente la cláusula 15 sobre retenciones?
```
**Cuándo usarla:** Necesitas encontrar información concreta rápidamente

### 2. Análisis Comparativo
```
Compara las obligaciones fiscales entre estos dos contratos
```
**Cuándo usarla:** Tienes múltiples documentos y necesitas comparar

### 3. Identificación de Patrones
```
¿Hay inconsistencias entre las cifras de estos documentos?
```
**Cuándo usarla:** Buscas problemas, errores o situaciones anómalas

### 4. Extracción Estructurada
```
Lista todas las cláusulas que mencionen impuestos en formato tabla
```
**Cuándo usarla:** Quieres un resumen organizado de elementos específicos

### 5. Síntesis
```
Resume los 5 riesgos fiscales principales de este contrato
```
**Cuándo usarla:** Necesitas el panorama general de alto nivel

## Ventajas de Claude Projects

1. **Análisis profundo:** Procesa hasta 200 páginas manteniendo coherencia completa
2. **Conocimiento persistente:** Los docs quedan guardados para futuras consultas
3. **Análisis cruzado:** Compara múltiples documentos simultáneamente
4. **Menos alucinaciones:** Especialmente bueno con docs técnicos y legales
5. **Razonamiento matizado:** Identifica contradicciones y ambigüedades
6. **Cita fuentes:** Referencia páginas y secciones específicas

---

## Cuándo Usar Claude Projects

**Usa Claude Projects cuando:**
- Tienes documentos de 50+ páginas
- Necesitas análisis profundo, no respuesta rápida
- Requieres análisis cruzado entre varios documentos largos
- El caso es único y complejo

**Usa Custom GPTs cuando:**
- Necesitas respuestas rápidas y consistentes
- La tarea es repetitiva
- Quieres compartir con tu equipo fácilmente
- Los documentos son cortos (menos de 30 páginas)

## Proceso de Creación (8 Pasos)

1. Acceder a Claude.ai con cuenta Pro
2. Click en "Projects" en menú lateral
3. "New Project"
4. Asignar nombre descriptivo
5. Agregar Custom Instructions
6. Subir documentos (Knowledge Base)
7. Crear primera conversación
8. Hacer preguntas y analizar

---

## Plantilla de Custom Instructions

```
CONTEXTO:
[Para qué es este proyecto]

ROL:
[Cómo debe comportarse Claude]

DOCUMENTOS INCLUIDOS:
[Lista de docs y su propósito]

AL RESPONDER SIEMPRE:
- [Instrucción 1]
- [Instrucción 2]
- [Instrucción 3]

FORMATO DE RESPUESTA:
[Estructura que prefieres]
```

---

## Mejores Prácticas

1. **Nombres descriptivos:** "Contrato Arrendamiento - Cliente XYZ - Enero 2024"
2. **Custom Instructions claras:** Define rol, contexto y formato esperado
3. **Orden lógico:** Sube docs principales primero, referencia después
4. **Pregunta amplia primero:** Obtén panorama general antes de lo específico
5. **Pide citas:** Siempre solicita página o sección de donde sacó info
6. **Proyectos por caso:** No por tipo de documento
7. **Ajusta instrucciones:** Si respuestas no son óptimas, refina y prueba de nuevo

---

## Seguridad y Confidencialidad

**Importante:**
- Los documentos se almacenan en servidores de Anthropic (Claude)
- No uses Projects para información ultra sensible sin autorización del cliente
- Lee los términos de servicio sobre uso de datos
- Considera implicaciones de secreto profesional
- Para casos muy sensibles, evalúa alternativas

**Buenas prácticas:**
- Anonimiza datos sensibles cuando sea posible
- Usa documentos públicos para práctica
- Ten autorización del cliente para subir sus docs
- Evalúa riesgo vs beneficio caso por caso

## Flujo de Trabajo Recomendado

1. **Recibir documentos** largos del cliente
2. **Crear Project** con nombre descriptivo (Cliente - Caso - Fecha)
3. **Subir documentos** y configurar Custom Instructions
4. **Primera pasada:** Pedir resumen general y puntos clave (mapa del doc)
5. **Análisis profundo:** Preguntas específicas según necesidad
6. **Documentar hallazgos** fuera de Claude (tu sistema de gestión)
7. **Validar** puntos críticos con lectura manual

---

## Combinando Herramientas

El poder está en usar la herramienta correcta para cada fase:

1. **Claude Project** → Análisis profundo inicial de documentos largos
2. **Custom GPT** → Consultas rápidas y repetitivas sobre ese tipo de caso
3. **Claude Project** → Siguiente caso único y complejo
4. **Custom GPT** → Estandarizar presentación de resultados

**No es uno u otro, es ambos según la situación.**

## Errores a Evitar

- Subir documentos sin Custom Instructions claras
- No pedir que cite página o sección específica

- No documentar hallazgos fuera de Claude
- Olvidar actualizar docs cuando cambien
