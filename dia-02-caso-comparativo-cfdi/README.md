## El Caso del Día: Lección Clave

En el caso práctico, el uso de CFDI era **G03 - Gastos en general** para mobiliario de oficina.

**El problema:** Mobiliario es activo fijo, debería usar **I02**.

**Implicaciones:**
- G03 = deducción inmediata como gasto
- I02 = deducción vía depreciación (10% anual)

**¿Quién lo identificó mejor?** Claude típicamente detecta estos matices técnicos.

## Prompts Clave del Día 2

### Análisis Rápido de CFDI
```
Analiza este CFDI y dime en 3 puntos:
1. ¿Está correctamente emitido?
2. ¿Es deducible?
3. ¿Qué debo verificar?

Datos: [pegar datos del CFDI]
```

### Verificar Uso de CFDI
```
¿El uso de CFDI [CÓDIGO] es correcto para [CONCEPTO]?
Si no es el correcto, ¿cuál debería ser y por qué?
```

### Respuesta para Cliente
```
Redacta un mensaje breve para mi cliente explicando
que su factura por [CONCEPTO] está/no está correcta.
Tono: profesional pero amigable.
```
