## Problemas Comunes y Soluciones

### Problema 1: "No se cual herramienta usar"
**Solucion:**
Usa el framework de preguntas:
1. ¿Trabajo con mis documentos o busco en internet?
2. ¿Necesito fuentes verificables?
3. ¿Es tarea recurrente o unica?
4. ¿Necesito integracion con Google?

### Problema 2: "Toma mucho tiempo cambiar entre herramientas"
**Solucion:**
- Ten las herramientas principales abiertas en pestañas fijas
- Crea bookmarks para acceso rapido
- Usa el mismo navegador para todas
- Con practica, el cambio se vuelve natural

### Problema 3: "No recuerdo que instrucciones darle a cada herramienta"
**Solucion:**
- Guarda tus prompts exitosos en un documento
- Usa Custom GPTs para tareas con instrucciones fijas
- Crea Projects en Claude con instrucciones persistentes
- Ten plantillas de prompts para cada tipo de tarea

### Problema 4: "Las respuestas de diferentes herramientas se contradicen"
**Solucion:**
- Perplexity y NotebookLM son para verificar
- Si hay contradiccion, ve a la fuente primaria (ley, DOF, SAT)
- Usa tu juicio profesional como arbitro final
- Documenta la contradiccion para investigar mas
