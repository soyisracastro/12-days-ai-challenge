# Dia 9: Workflow Definitivo

## Resumen

El Dia 9 cierra la Semana 3 integrando todas las herramientas aprendidas en un sistema de trabajo coherente. La clave no es usar mas herramientas, sino usar la correcta en el momento correcto. Este dia enseÃ±a a crear workflows personalizados que combinan ChatGPT, Claude, Gemini, NotebookLM y Perplexity segun la tarea.

## Puntos Clave

- **Framework de decision:** 5 preguntas para elegir la herramienta correcta
- **Workflows tipicos:** Investigacion, analisis de documentos, tareas recurrentes
- **Regla de oro:** La herramienta correcta > la herramienta favorita
- **Sistema personalizado:** Cada contador crea su propio workflow
- **Verificacion siempre:** Ninguna IA reemplaza la validacion humana

## Matriz de Seleccion

| Necesidad | Herramienta |
|-----------|-------------|
| Buscar info con fuentes | Perplexity |
| Consultar mis documentos | NotebookLM |
| Tareas recurrentes | Custom GPTs |
| Documentos largos | Claude Projects |
| Google Workspace | Gemini |
| Tarea general unica | ChatGPT o Claude |

## Archivos de Este Dia

- `slides-script.md` - Slides con script para teleprompter
- `ejercicios-practicos.md` - Ejercicios para 30-45 min de practica
- `contenido-plataforma.md` - Contenido para plataforma Nas.io

## Flujos de Trabajo Tipicos

1. **Investigacion fiscal:** Perplexity -> NotebookLM -> Claude (redaccion)
2. **Analisis de documentos:** Claude Projects -> NotebookLM -> ChatGPT (informe)
3. **Tareas recurrentes:** Custom GPT -> Verificacion -> Documentar
4. **Preparar capacitacion:** NotebookLM -> Perplexity -> Gemini -> Claude
5. **Due diligence:** Claude Projects -> NotebookLM -> Perplexity -> ChatGPT

## Duracion

- Teoria: 25-30 minutos
- Practica: 30-45 minutos
- Total: 60-75 minutos
