## Reto 12 Días de IA para Contadores

## Resumen del Día

Hoy aprendiste a crear tus propios asistentes especializados sin programar. Los **Custom GPTs** son como empleados virtuales entrenados específicamente para las tareas que tú defines.

Ya no necesitas repetir el mismo prompt cada vez. Creas el GPT una vez, lo compartes con tu equipo, y todos obtienen respuestas consistentes y especializadas.

---

## ¿Qué es un Custom GPT?

Un Custom GPT es una versión personalizada de ChatGPT que:

- Tiene instrucciones permanentes sobre cómo comportarse
- Incluye documentos de referencia precargados (leyes, reglamentos, guías)
- Responde siempre de forma consistente
- Se puede compartir con tu equipo o clientes
- No requiere conocimientos de programación

**Es como contratar a un especialista que ya conoce el contexto y procedimientos de tu despacho.**

---

## Componentes de un Custom GPT

Todo Custom GPT se compone de:

| Componente | Descripción |
|------------|-------------|
| **Nombre y descripción** | Identifica para qué sirve el GPT |
| **Instructions** | El prompt permanente que define su comportamiento |
| **Conversation starters** | Preguntas sugeridas para empezar |
| **Knowledge** | Archivos de referencia (leyes, guías, hasta 20 archivos) |
| **Capabilities** | Funciones especiales: cálculos, navegación web, imágenes |

---

## Casos de Uso para Contadores

Creamos ejemplos de tres Custom GPTs:

### 1. Asesor RESICO México
- **Para qué:** Resolver dudas sobre Régimen Simplificado de Confianza
- **Conocimiento:** Título IV.A LISR, RMF 2024, tablas de tasas
- **Ahorra:** 15-20 min por consulta de cliente

### 2. Validador CFDI 4.0
- **Para qué:** Revisar si un comprobante cumple requisitos técnicos y fiscales
- **Conocimiento:** Guía de llenado SAT, Anexo 20, catálogos
- **Ahorra:** 30 min de revisión manual por CFDI complejo

### 3. Calculadora de Nómina
- **Para qué:** Calcular ISR, IMSS, INFONAVIT y neto a pagar
- **Conocimiento:** Tablas ISR, tarifas IMSS, LSS
- **Ahorra:** 20 min por cálculo de recibo completo

---

## Ventajas de los Custom GPTs

1. **Consistencia:** Todos en tu despacho usan el mismo criterio
2. **Especialización:** Es experto en un tema específico, no generalista
3. **Ahorro de tiempo:** No repites contexto en cada consulta
4. **Escalabilidad:** Lo creas una vez, lo usa todo el equipo
5. **Actualizable:** Cambias instrucciones cuando cambien las leyes
6. **Sin código:** Si sabes crear un prompt, puedes crear un GPT

## Fórmula de Instrucciones Efectivas

Usa la misma estructura de prompts del Día 3, pero ahora es permanente:

```
ROL: ¿Qué tipo de experto es?
OBJETIVO: ¿Qué busca lograr el usuario?
CONOCIMIENTO: ¿Qué temas domina?
FORMATO: ¿Cómo estructura las respuestas?
LÍMITES: ¿Qué NO debe hacer?
```

---

## Custom GPTs vs ChatGPT Normal

| Aspecto | ChatGPT Normal | Custom GPT |
|---------|---------------|------------|
| Contexto | Se pierde entre sesiones | Permanente |
| Especialización | Generalista | Experto en un tema |
| Archivos | Subes cada vez | Precargados |
| Compartir | Copias el prompt | Compartes el link |
| Consistencia | Varía según quien escriba | Siempre igual |

---

## Mejores Prácticas

1. **Prueba exhaustivamente** antes de compartir con clientes
2. **Actualiza cuando cambien leyes** fiscales (calendario de revisión)
3. **Incluye disclaimers** sobre responsabilidad profesional
4. **Usa conversation starters** para guiar al usuario
5. **Sube documentos oficiales**, no resúmenes de internet
6. **Pide feedback** y mejora continuamente

Al publicar tu Custom GPT tienes tres opciones:

2. **Anyone with a link:** Cualquiera con el link (ideal para equipo)

## Limitaciones a Considerar

- Requiere ChatGPT Plus (20 USD/mes)
- Máximo 20 archivos de conocimiento
- Puede olvidar instrucciones en conversaciones muy largas (raro)
- No reemplaza el criterio profesional del contador
- Requiere mantenimiento cuando cambien leyes

## Errores a Evitar

- Crear GPTs genéricos sin especialización clara
- No probar suficientemente antes de compartir
- Olvidar actualizar cuando cambien leyes
- Confiar ciegamente sin validar respuestas críticas
- No incluir disclaimers de responsabilidad profesional
- Usar solo una herramienta para todo (combina GPTs según la tarea)
