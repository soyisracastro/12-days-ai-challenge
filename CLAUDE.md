# Contexto del Proyecto: Reto 12 Días de IA para Contadores

## Descripción General

Este proyecto contiene el contenido completo para un reto/taller de 12 días sobre Inteligencia Artificial para contadores. El contenido se publica en **TodoConta.com** y se entrega a los participantes vía **Notion/Docs**.

**Landing page:** https://todoconta.com/ia-contadores/reto-12-dias

---

## Estructura del Reto

### Semana 1: Fundamentos (Días 1-3) ✅ COMPLETADA
- **Día 1:** Mentalidad IA-First - Marco de decisión para elegir herramientas
- **Día 2:** Caso Comparativo CFDI 4.0 - Mismo problema con ChatGPT, Claude y Gemini
- **Día 3:** Prompts Universales - Fórmula ROL+CONTEXTO+TAREA+FORMATO + 10 plantillas

### Semana 2: Especialización (Días 4-6) ✅ COMPLETADA
- **Día 4:** Custom GPTs Fiscales - Asistentes especializados sin código (RESICO, nómina, CFDI)
- **Día 5:** Claude Projects - Análisis de contratos, dictámenes y normativa extensa
- **Día 6:** Gemini + Google Workspace - Conexión con Drive, Sheets, Docs

### Semana 3: Herramientas Avanzadas (Días 7-9)
- **Día 7:** NotebookLM - Notebooks con LISR, CFF y criterios SAT (RAG)
- **Día 8:** Perplexity Pro - Búsqueda en tiempo real con fuentes citadas
- **Día 9:** Workflow Definitivo - Flujos híbridos combinando herramientas

### Semana 4: Aplicación y Negocio (Días 10-12)
- **Día 10:** Caso Multimodal - Procesamiento de múltiples formatos, auditorías
- **Día 11:** Diseña tu Sitio Web - HTML/CSS sin programación, hosting gratuito
- **Día 12:** Monetización - 4 modelos de negocio con IA

---

## Formato de Entregables por Día

Cada día debe incluir estos archivos en su carpeta:

```
dia-XX-nombre/
├── slides-script.md      # Slides con script para teleprompter
├── [recursos].md         # Material específico del día (matrices, guías, plantillas)
├── ejercicios-practicos.md  # Ejercicios para 30-45 min de práctica
└── README.md             # Resumen para página de Notion
```

### Especificaciones de Contenido

| Elemento | Especificación |
|----------|----------------|
| **Duración teoría** | 20-30 minutos |
| **Duración práctica** | 30-45 minutos |
| **Nivel participantes** | Principiantes (nunca/casi nunca han usado IA) |
| **Formato slides** | Markdown estructurado (para Canva/PowerPoint) |
| **Entrega material** | Notion/Docs |
| **Script** | Listo para leer en teleprompter |
