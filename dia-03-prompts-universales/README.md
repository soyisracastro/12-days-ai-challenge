## Resumen del Día

Hoy aprendiste la fórmula definitiva para crear prompts efectivos y te llevaste 10 plantillas listas para usar. No más empezar desde cero, no más respuestas genéricas.

---

## La Fórmula Mágica

```
ROL + CONTEXTO + TAREA + FORMATO = Prompt Efectivo
```

| Elemento | Propósito | Ejemplo |
|----------|-----------|---------|
| **ROL** | Define la perspectiva | "Actúa como contador fiscal en México" |
| **CONTEXTO** | Proporciona información | "Mi cliente es PF RESICO..." |
| **TAREA** | Especifica la acción | "Analiza si es deducible..." |
| **FORMATO** | Define la presentación | "Lista numerada con fundamento legal" |

## Ejemplo Antes vs Después

### Antes (sin fórmula):
```
¿Puede mi cliente deducir su auto?
```
Resultado: Respuesta genérica e incompleta.

### Después (con fórmula):
```
Actúa como contador fiscal especialista en deducciones en México.

CONTEXTO:
- Cliente: PF actividad empresarial, régimen general
- Compró auto de $600,000 para trabajo (70% uso negocio)

ANALIZA:
1. ¿Puede deducirlo?
2. ¿Cuál es el límite?
3. ¿Qué requisitos debe cumplir?

FORMATO: Lista con fundamento legal.
```
Resultado: Respuesta específica, útil y con fundamento.

## Potenciadores

Agrega al final de cualquier prompt:

- "Sé específico y práctico"
- "Incluye fundamento legal"
- "Dame la postura más conservadora fiscalmente"
- "Si hay ambigüedad, indícalo"
