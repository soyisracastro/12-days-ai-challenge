# Las 10 Plantillas de Prompts para Contadores
## Listas para Copiar, Pegar y Adaptar

---

## Cómo Usar Este Documento

1. **Copia** la plantilla que necesites
2. **Reemplaza** todo lo que está entre [CORCHETES] con tus datos
3. **Elimina** las líneas que no apliquen a tu caso
4. **Agrega** información adicional si es necesario
5. **Pega** en ChatGPT, Claude o Gemini

## Plantilla 1: Consulta Fiscal General

**Uso:** Cualquier duda fiscal general
**Mejor herramienta:** Claude o ChatGPT

```
Actúa como contador fiscal especialista en [ÁREA: impuestos/nómina/comercio exterior] en México.

CONTEXTO:
- Tipo de contribuyente: [PERSONA FÍSICA / PERSONA MORAL]
- Régimen fiscal: [RESICO / ACTIVIDAD EMPRESARIAL / GENERAL DE LEY PM / OTRO]
- Giro o actividad: [DESCRIPCIÓN DEL NEGOCIO]
- Situación específica: [DESCRIBE LA SITUACIÓN O ANTECEDENTES]

CONSULTA:
[ESCRIBE TU PREGUNTA ESPECÍFICA]

NECESITO QUE ME INDIQUES:
1. Respuesta directa a la consulta
2. Fundamento legal aplicable (artículos de ley)
3. Requisitos o condiciones que deben cumplirse
4. Recomendaciones prácticas
5. Riesgos o consideraciones especiales

Sé específico y considera el contexto fiscal mexicano vigente.
```

## Plantilla 2: Análisis de Deducibilidad

**Uso:** Determinar si un gasto es deducible
**Mejor herramienta:** Claude (análisis profundo) o ChatGPT (respuesta rápida)

```
Actúa como contador fiscal especialista en deducciones fiscales en México.

MI CLIENTE:
- Tipo: [PERSONA FÍSICA / PERSONA MORAL]
- Régimen fiscal: [RÉGIMEN]
- Giro o actividad: [DESCRIPCIÓN]
- Ingresos anuales aproximados: $[MONTO] (opcional)

EL GASTO EN CUESTIÓN:
- Concepto: [DESCRIPCIÓN DETALLADA DEL GASTO]
- Monto: $[CANTIDAD]
- Frecuencia: [PAGO ÚNICO / MENSUAL / ANUAL]
- Proveedor: [TIPO: NACIONAL/EXTRANJERO, FORMAL/INFORMAL]
- Relación con la actividad: [CÓMO SE RELACIONA CON EL NEGOCIO]

ANALIZA:
1. ¿Es deducible este gasto? ¿Bajo qué condiciones?
2. ¿Hay límites o topes de deducción aplicables?
3. ¿Qué porcentaje es deducible (si aplica)?
4. ¿Qué requisitos documentales debe cumplir (CFDI, contratos, etc.)?
5. ¿Cuál es el fundamento legal? (Artículos de LISR)
6. ¿Hay riesgos o consideraciones especiales?

Formato: Lista numerada con fundamento legal en cada punto.
```

[DESCRIBE QUÉ NECESITAS INFORMAR AL CLIENTE]

- Nombre/Empresa: [NOMBRE]

- Nivel de conocimiento fiscal: [NULO / BÁSICO / MEDIO / ALTO]
- Relación: [FORMAL / CERCANA]

EL CORREO DEBE:
- Explicar: [EL PUNTO PRINCIPAL]
- Incluir: [DATOS ESPECÍFICOS SI APLICA: MONTOS, FECHAS, ETC.]

- Longitud: [CORTO: 1-2 PÁRRAFOS / MEDIO: 3-4 PÁRRAFOS / LARGO]

Genera:
1. Línea de asunto sugerida
2. Cuerpo del correo
3. Despedida apropiada

## Plantilla 4: Cálculo con Explicación

**Uso:** Cálculos fiscales con procedimiento paso a paso
**Mejor herramienta:** ChatGPT

```
Actúa como contador fiscal en México especializado en [TIPO DE CÁLCULO: ISR / IVA / NÓMINA / IMSS / OTRO].

DATOS PARA EL CÁLCULO:
- [DATO 1]: $[VALOR]
- [DATO 2]: $[VALOR]
- [DATO 3]: $[VALOR]
- [AGREGAR MÁS DATOS SEGÚN NECESITES]
- Período: [MES Y AÑO]
- Tipo de contribuyente: [PF/PM]
- Régimen: [RÉGIMEN]

CALCULA:
[DESCRIBE ESPECÍFICAMENTE QUÉ QUIERES CALCULAR]

MUESTRA:
1. El procedimiento paso a paso
2. La fórmula o tarifa aplicada en cada paso
3. El fundamento legal (artículos de ley o tablas aplicables)
4. El resultado final
5. Una verificación o comprobación del cálculo

El formato debe ser claro y detallado para poder:
- Validar que el cálculo esté correcto
- Explicarlo al cliente si es necesario
```

## Plantilla 5: Análisis de Documento/Contrato

**Uso:** Revisar contratos, convenios, actas
**Mejor herramienta:** Claude (puede procesar documentos largos)

```
Actúa como [CONTADOR FISCAL / ABOGADO FISCALISTA] especializado en análisis de [TIPO DE DOCUMENTO: CONTRATOS / CONVENIOS / ACTAS / DICTÁMENES] en México.

DOCUMENTO A ANALIZAR:
[PEGA EL TEXTO DEL DOCUMENTO O DESCRIBE SU CONTENIDO]
[SI USAS CLAUDE, PUEDES SUBIR EL ARCHIVO PDF]

CONTEXTO:
- Mi cliente es: [EMISOR / RECEPTOR / UNA DE LAS PARTES]
- Tipo de cliente: [PF/PM] en régimen [RÉGIMEN]
- Objetivo de la revisión: [QUÉ BUSCA EL CLIENTE: FIRMAR / RENEGOCIAR / ENTENDER OBLIGACIONES]

ANALIZA E IDENTIFICA:
1. RESUMEN EJECUTIVO (máximo 1 párrafo)
   - De qué trata el documento
   - Partes involucradas

2. IMPLICACIONES FISCALES
   - Obligaciones fiscales para mi cliente
   - Retenciones aplicables (ISR, IVA)
   - Momentos de causación de impuestos

3. RIESGOS Y CONTINGENCIAS
   - Cláusulas problemáticas
   - Riesgos fiscales potenciales
   - Ambigüedades que podrían causar problemas

4. DOCUMENTACIÓN RELACIONADA
   - CFDI que se generarán
   - Contratos o documentos complementarios necesarios

5. RECOMENDACIONES
   - Modificaciones sugeridas
   - Puntos a negociar
   - Acciones a tomar

Si encuentras información faltante o ambigua, indícalo claramente.
```

## Plantilla 6: Respuesta a Requerimiento del SAT

**Uso:** Analizar y preparar respuesta a requerimientos
**Mejor herramienta:** Claude (por precisión técnica)

```
Actúa como contador fiscal especializado en procedimientos administrativos y defensa fiscal ante el SAT en México.

EL REQUERIMIENTO:
- Tipo de documento: [CARTA INVITACIÓN / REQUERIMIENTO DE INFORMACIÓN / REQUERIMIENTO DE OBLIGACIONES / ACTA DE VISITA / OTRO]
- Autoridad emisora: [SAT / ADMINISTRACIÓN LOCAL / AGAFF]
- Número de folio u oficio: [NÚMERO SI LO TIENES]
- Fecha del documento: [FECHA]
- Fecha de notificación: [FECHA EN QUE SE RECIBIÓ]

LO QUE SOLICITA EL SAT:
[DESCRIBE O LISTA LO QUE ESTÁN SOLICITANDO]

SITUACIÓN DEL CONTRIBUYENTE:
- Tipo: [PERSONA FÍSICA / PERSONA MORAL]
- Régimen fiscal: [RÉGIMEN]
- RFC: [RFC - OPCIONAL]
- Antecedentes relevantes: [CUALQUIER CONTEXTO IMPORTANTE]
- ¿Existe la irregularidad que señalan?: [SÍ / NO / PARCIALMENTE]

NECESITO:
1. ANÁLISIS DEL REQUERIMIENTO
   - Qué solicita exactamente el SAT
   - Fundamento legal del requerimiento

2. PLAZOS
   - Plazo legal para responder
   - Fecha límite considerando días hábiles

3. DOCUMENTACIÓN A PREPARAR
   - Lista de documentos que debemos reunir
   - Formatos o requisitos específicos

4. BORRADOR DE ESCRITO DE RESPUESTA
   - Estructura del oficio de respuesta
   - Argumentos a incluir
   - Fundamentación legal

5. ESTRATEGIA RECOMENDADA
   - Cómo abordar la respuesta
   - Qué incluir y qué evitar

6. RIESGOS
   - Consecuencias de no atender
   - Multas o sanciones aplicables
   - Siguientes pasos del SAT si no se satisface

Sé preciso con plazos y fundamentación legal.
```

## Plantilla 7: Comparativo de Opciones Fiscales

**Uso:** Comparar alternativas para tomar decisiones
**Mejor herramienta:** ChatGPT (tablas claras) o Claude (análisis profundo)

```
Actúa como asesor fiscal estratégico en México.

MI CLIENTE:
- Perfil: [DESCRIPCIÓN GENERAL DEL CLIENTE]
- Situación actual: [CÓMO ESTÁ TRIBUTANDO AHORA / SITUACIÓN PRESENTE]
- Ingresos estimados: $[MONTO ANUAL]
- Objetivo: [QUÉ QUIERE LOGRAR: OPTIMIZAR CARGA FISCAL / SIMPLIFICAR / CRECER / OTRO]

OPCIONES A COMPARAR:
- OPCIÓN A: [DESCRIBE LA PRIMERA OPCIÓN]
- OPCIÓN B: [DESCRIBE LA SEGUNDA OPCIÓN]
- OPCIÓN C: [DESCRIBE LA TERCERA OPCIÓN - SI APLICA]

COMPARA CONSIDERANDO:
1. Carga fiscal estimada de cada opción
2. Obligaciones fiscales y administrativas
3. Complejidad contable
4. Costos de implementación y mantenimiento
5. Flexibilidad y escalabilidad futura
6. Riesgos y contingencias de cada una
7. Requisitos para cambiar de una a otra

FORMATO DE RESPUESTA:
1. Tabla comparativa con todos los criterios
2. Análisis de pros y contras de cada opción
3. Recomendación final con justificación
4. Pasos para implementar la opción recomendada
```

## Plantilla 8: Explicación para Cliente No Técnico

**Uso:** Traducir conceptos fiscales a lenguaje simple
**Mejor herramienta:** ChatGPT

```
Actúa como contador que sabe explicar temas fiscales complejos de forma simple y accesible para personas sin conocimientos técnicos.

CONCEPTO A EXPLICAR:
[TEMA FISCAL QUE NECESITAS EXPLICAR: RESICO / RETENCIONES / CFDI / DECLARACIÓN ANUAL / OTRO]

CONTEXTO DEL CLIENTE:
- Perfil: [EMPLEADO / PEQUEÑO EMPRESARIO / PROFESIONISTA / OTRO]
- Nivel de conocimiento fiscal: [NULO / MUY BÁSICO]
- Por qué necesita entenderlo: [SITUACIÓN ESPECÍFICA]

GENERA UNA EXPLICACIÓN QUE:
1. Use lenguaje cotidiano (cero tecnicismos)
2. Use analogías o ejemplos de la vida diaria
3. Explique POR QUÉ le importa esto al cliente (el impacto personal)
4. Sea breve (máximo 3-4 párrafos cortos)
5. Incluya qué debe HACER el cliente (acciones concretas)
6. Termine con un resumen de "puntos clave" en bullets

Imagina que se lo explicas a tu mamá o a un amigo que no sabe nada de impuestos.

EVITA:
- Artículos de ley (menciona solo si es absolutamente necesario)
- Porcentajes complejos
- Siglas sin explicar
- Asumir conocimiento previo
```

## Plantilla 9: Revisión de CFDI

**Uso:** Validar facturas recibidas
**Mejor herramienta:** ChatGPT (rápido) o Claude (análisis profundo)

```
Actúa como contador especialista en CFDI 4.0 en México.

DATOS DEL CFDI A REVISAR:

EMISOR:
- Razón Social: [NOMBRE DEL PROVEEDOR]
- RFC: [RFC DEL EMISOR]
- Régimen Fiscal: [RÉGIMEN DEL EMISOR]

RECEPTOR (MI CLIENTE):
- Razón Social: [NOMBRE]
- RFC: [RFC]
- Régimen Fiscal: [RÉGIMEN]
- Giro o actividad: [A QUÉ SE DEDICA]

DATOS DE LA FACTURA:
- Uso de CFDI: [CÓDIGO Y DESCRIPCIÓN]
- Concepto(s): [DESCRIPCIÓN DE LO QUE SE COMPRÓ/CONTRATÓ]
- Subtotal: $[MONTO]
- IVA: $[MONTO]
- Total: $[MONTO]
- Método de pago: [PUE / PPD]
- Forma de pago: [CÓDIGO Y DESCRIPCIÓN]

CONTEXTO ADICIONAL:
- ¿Primera vez con este proveedor?: [SÍ/NO]
- ¿Monto inusual para este tipo de compra?: [SÍ/NO]
- ¿Ya se pagó?: [SÍ/NO/PARCIALMENTE]

REVISA Y RESPONDE:
1. ¿El CFDI cumple con los requisitos de CFDI 4.0?
2. ¿El uso de CFDI es correcto para este concepto?
3. ¿El método y forma de pago son coherentes?
4. ¿Es deducible para mi cliente? ¿Cómo tratarlo fiscalmente?
5. ¿Qué banderas rojas o puntos de atención identificas?
6. ¿Qué documentación complementaria se necesita?
7. ¿Se requiere complemento de pago? ¿Cuándo?

Sé específico y práctico en tus observaciones.
```

## Plantilla 10: Investigación de Criterio Fiscal

**Uso:** Buscar fundamento legal y criterios oficiales
**Mejor herramienta:** Gemini (información actualizada)

```
Actúa como investigador fiscal especializado en normativa tributaria mexicana.

TEMA A INVESTIGAR:
[DESCRIBE LA DUDA O SITUACIÓN FISCAL QUE QUIERES FUNDAMENTAR]

CONTEXTO:
- ¿Para quién es?: [TIPO DE CONTRIBUYENTE]
- ¿Por qué surge la duda?: [SITUACIÓN ESPECÍFICA]
- ¿Qué año fiscal aplica?: [AÑO]

BUSCA Y PROPORCIONA:

1. FUNDAMENTO LEGAL PRINCIPAL
   - Ley aplicable (LISR, LIVA, CFF, LGSM, etc.)
   - Artículos específicos con su texto relevante

2. REGLAS DE RESOLUCIÓN MISCELÁNEA
   - Reglas de la RMF relacionadas
   - Vigencia de las reglas

3. CRITERIOS DEL SAT
   - Criterios normativos o no vinculativos
   - Postura oficial del SAT sobre el tema

4. JURISPRUDENCIA
   - Tesis o jurisprudencias de la SCJN o TFJA
   - Precedentes relevantes

5. INTERPRETACIÓN PRÁCTICA
   - Cómo aplicar esto en la práctica
   - Zonas grises o ambigüedades
   - Postura conservadora vs agresiva

IMPORTANTE:
- Si tu información puede estar desactualizada, indícalo
- Si hay interpretaciones distintas, menciona las diferentes posturas
- Cita las fuentes específicas (número de artículo, fecha de publicación, etc.)
- Indica si debo verificar en fuentes oficiales actuales
```

## Potenciadores para Agregar a Cualquier Plantilla

Agrega estas frases al final de cualquier prompt para mejorar resultados:

### Para mayor precisión:
```
- "Sé específico y práctico en tus recomendaciones"
- "Si hay ambigüedad o zonas grises, indícalo claramente"
- "Piensa paso a paso antes de responder"
```

### Para fundamentación:
```
- "Incluye fundamento legal (artículos de ley) en cada punto relevante"
- "Cita las fuentes específicas cuando sea posible"
```

### Para seguridad fiscal:
```
- "Dame la postura más conservadora fiscalmente"
- "Indica los riesgos de cada opción"
- "Señala si el SAT podría cuestionar algo"
```

### Para información actualizada:
```
- "Considera que estamos en [AÑO]"
- "Si tu información puede no estar actualizada, indícalo"
- "Sugiere verificar en fuentes oficiales del SAT"
```

### Para honestidad:
```
- "Si no tienes certeza, dilo en lugar de adivinar"
- "Si te falta información para responder bien, pregúntame"
```
